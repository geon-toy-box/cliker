# 콘텐츠 등급 (IARC) 설문 답변 가이드 — 클리커 (cliker)

> Google Play Console → 앱 콘텐츠 → **콘텐츠 등급(Content rating)** 설문(IARC) 작성용.
> 앱에는 폭력·성적 표현·욕설·도박·약물 등 어떤 유해 콘텐츠도 없으므로, 아래대로
> 모두 "아니요"를 선택하면 된다. **예상 등급: 전체이용가 / Everyone.**

---

## 1. 카테고리 선택

- 설문 시작 시 앱 카테고리를 묻는다. 본 앱은 게임이 아니므로 **"기타 모든 앱 유형
  (All other app types)"** / 또는 "참조, 뉴스 또는 교육용" 이 아닌 **유틸리티/오락**
  성격에 맞는 일반 앱 카테고리를 선택한다. (게임 설문이 아니라 일반 앱 설문으로 진행.)

## 2. 설문 문항별 답변 (모두 "아니요")

| 주제 | 본 앱에 해당? | 답변 |
|------|----------------|------|
| 폭력 (Violence) — 사실적/만화적 폭력, 유혈 | 없음 | **아니요 (No)** |
| 성적 콘텐츠 (Sexuality / Nudity) | 없음 | **아니요 (No)** |
| 욕설/비속어 (Profanity / Crude humor) | 없음 | **아니요 (No)** |
| 약물·음주·담배 (Controlled substances) | 없음 | **아니요 (No)** |
| 도박 (Gambling / Simulated gambling) | 없음 | **아니요 (No)** |
| 공포/무서운 콘텐츠 (Fear / Horror) | 없음 | **아니요 (No)** |
| 사용자 간 소통/콘텐츠 공유 (User interaction, UGC, chat) | 없음(완전 오프라인, 네트워크 없음) | **아니요 (No)** |
| 위치 공유 (Shares location) | 없음 | **아니요 (No)** |
| 디지털 구매/유료 콘텐츠 (Digital purchases) | 없음(무료, 인앱결제 없음) | **아니요 (No)** |
| 개인정보 공유 (Shares personal info) | 없음 | **아니요 (No)** |

## 3. 앱 내용 한 줄 설명 (설문에서 요구 시)

> "화면 속 기계식 키캡을 눌러 타건음·진동·LED 효과로 스트레스를 푸는 단순 클릭커
> 앱. 유해 콘텐츠 없음, 광고·결제·네트워크 없음."

## 4. 예상 결과

- IARC 등급: **전체이용가 (Everyone / 3+)**
- 위 답변에 유해 요소가 하나도 없으므로 각 등급기관(ESRB/PEGI/USK/GRAC 등)에서
  최저 연령 등급이 부여될 것으로 예상된다.
- ⚠️ 최종 등급은 IARC 설문 제출 후 자동 산정된다. 본 문서는 **예상치**이며, 실제
  등급 결정은 Play/IARC의 몫이다(사용자 제출 필요).
